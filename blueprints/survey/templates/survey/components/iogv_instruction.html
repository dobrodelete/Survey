Инструкция по заполнению

Уважаемы респондент!

Вопросы по оценке цифровой зрелости разделены на шесть направлений:

  1. Организационная культура и управление
  2. Кадры
  3. Процессы
  4. Продукты
  5. Данные
  6. Инфраструктура и инструменты

Всего в сумме 66 вопросов, по каждому из вопросов необходимо проставить оценку в активную ячейку. Модель оценки – дискретная от 0 до 4 с шагом 0,25.

Для точного выбора значения используется шкала с разбивкой по уровням ( Для каждого из уровней предложен вариант ответа).

Вначале необходимо определиться с уровнем, который наиболее подходит для вашего ИОГВ, затем - выставить ему оценку в столбец «Ранг(ваша оценка)».

Пример выбора оценки для уровня «1-2»:

Критерий Ранг (ваша оценка) Уровень цифровой зрелости
0-1 1-2 2-3 3-4
